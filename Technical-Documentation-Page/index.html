Introduction

Python is a simple, readable and versatile interpreted programming language. It is widely used in web development, data science, artificial intelligence, cybersecurity, automation and many other fields.

Why learn Python?

  • Easy for beginners to learn
  • Clear, human-like syntax
  • Large community and many resources
  • Compatible with all operating systems
Installation

Steps:

  1. Go to the : official website
  2. Download the latest version of Python (3.x)
  3. Install it by checking the “Add Python to PATH” option
  4. In a terminal or command prompt, type :
python --version
Basic Syntax

Display text:

print("Hello World")

Creating variables:

name = "Alice"
age = 25
is_major = True

Comments:

# This is a comment
Control Structures

Condition if/else:

age = 17
if age >= 17:
print("You are an adult.")
else:
print("You are a minor.")

Loop for:

for i in range(5):
print("Compteur :", i)

Loop while:

i = 0
while i < 3:
print("i vaut", i)
i += 1
Functions

Defining and calling a function :

def dire_bonjour(nom):
print("Bonjour", nom)
dire_bonjour("Alice")
Lists & Dictionaries

Lists:

fruits = ["pomme", "banane", "mangue"]
print(fruits[0]) # Affiche "pomme"

Dictionaries

personne = {
"nom": "Alice",
"âge": 25
}
print(personne["nom"]) # Affiche "Alice"
Tips & Best Practices
  • Always indent your code (Python uses indentation to organize blocks)
  • Choose clear and explicit variable names
  • Test your code often
  • Comment when useful
